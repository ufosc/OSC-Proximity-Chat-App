# Echo Chat App (formerly OSC Proximity Chat App)

![banner](/client/assets/images/github/Echo_banner_small2.png)

Echo is a new chat messaging app aimed at allowing you to talk to strangers _(or as we like to say, friends who haven't met ğŸ˜†)_ within a radius around you. This allows for more time/location relevant chats as well as allowing users to connect with the people who are closer to them. Unlike many chat apps, our app aims to keep interactions in real time and doesn't save chats/information after you are done chatting! Just like real conversations, we believe the best socializing happens on the spot!
